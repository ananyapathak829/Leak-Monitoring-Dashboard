# Leak Monitoring Dashboard

A full-stack MERN application for monitoring industrial sensor data through an interactive dashboard.

## Features
- Sensor monitoring dashboard
- Real-time sensor data display
- Node details page
- REST API backend
- MongoDB database integration
- Responsive React frontend

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- REST APIs
- JavaScript

## Getting Started

### Backend
cd backend2
npm install
npm start

### Frontend
cd Frontend2
npm install
npm run dev
